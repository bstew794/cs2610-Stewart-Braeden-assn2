Git Commands:

Git Add; this prepares a file for Git Commit:
git add [file]

Git Commit; this saves file changes to the repository as a version to later be pushed to a remote origin, along with a message:
git commit -m [message]

Git Status; this checks the working directory for file changes:
git status

Git Push; this pushes the staged commits to remote repository:
git push -u origin

CSS Basics:

background-color: this changes the background for the HTML element (.class, or #id) it is linked to:
[object] {background-color: [color hex code];}

color: this changes the color of the text of the object is linked to:
[object] {color: [color hex code];}

font-family: this changes the font family for the object's text:
[object] {font-family: [font family];}

font-style: this changes the style of the font (underlined, italicized, etc):
[object] {font-style: [font style];}

text-align: determines text alignment within the object:
[object] {text-align: [alignment direction];}

margin: dictates how many pixels of space an object should have between itself and other objects:
[object] {margin: [quantity of pixels]px;}

padding: pads items within the object:
[object] {padding: {quantity of pixels]px;}
